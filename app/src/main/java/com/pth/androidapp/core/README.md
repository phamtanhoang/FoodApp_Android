# Core Layer

## Description
Core layer contains the basic and common components of the application, used throughout all other layers.

## Structure
- **base/**: Base classes for Activity, Fragment, ViewModel
- **common/**: Common utility and helper classes
- **di/**: Dependency Injection modules
- **extensions/**: Kotlin extension functions
- **network/**: Network utilities and base classes
- **security/**: Security utilities and encryption
- **utils/**: Common utility classes

## Usage
Core layer is imported and used by all other layers in the application.
